MLBPARKER 주요 변경사항

v1.2.1

  • 자동로그인 기능 추가(로그인 팝업창)
  • 회원 메모 기능 추가
  • 다크모드 추가
  • 댓글 수정 시 텍스트 유지
  • 캐시 삭제를 통한 앱 최적화 기능 추가

v1.2.0

  • 이미지 확대/저장 기능 추가(이미지 클릭)
  • 유저 게시물, 가입일 보기 추가(아바타 클릭)
  • 건의사항 게시판 추가

v1.1.9

  • 버그 수정 및 안정화

v1.1.8

  • 키워드 필터링 기능 추가

v1.1.7

  • 말머리 필터링 기능 추가
  • 이미지 메뉴 추가(확대보기, 링크 복사)

v1.1.6

  • 게시판 당겨서 새로고침 추가 - 말머리 현행화(키움)

v1.1.5

  • 글 작성시 이미지 첨부 기능 추가(스토리지/SD카드 접근 권한 추가됨)

v1.1.3

  • 슈퍼 블로킹 기능 추가(유저 아바타 클릭)
    • 슈퍼블로킹: 로그인 없이 특정 유저의 글, 리플 차단 (게시판, 담장)
